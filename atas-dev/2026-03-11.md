## 🔍 ANÁLISE DOS PROBLEMAS

### Problema A: Frontend Instability
- Frontend inicia rapidamente mas pode estar em "building" mode
- Vite (dev server) pode estar ainda compilando quando browser tenta acessar
- **Solução sugerida**: Aumentar aguarde no TPIC para >30s antes de abrir browser

### Problema B: Browser Fecha Prematuramente  
- "Target page, context or browser has been closed" indica que o browser foi fechado
- Possível: Timeout curto, browser crash, ou memória insuficiente
- **Solução sugerida**: Adicionar tratamento de reconexão

### Problema C: Intents Não Encontradas
- Intent "admin_site" não encontrada na homepage
- Intent "paroquias" não encontrada após login
- **Possível causa**: Frontend em build mode, página JavaScript ainda carregando
- **Solução sugerida**: Aguardar window.location estar pronto antes de procurar elementos
